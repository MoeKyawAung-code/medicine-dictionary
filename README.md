# 💊 Medicine Dictionary · ဆေးဝါးအဘိဓာန်

A bilingual **English & Burmese** medicine reference dictionary, deployable as a single HTML file on GitHub Pages.

## ✨ Features

- 🔍 **Real-time search** in English & Burmese
- 🗂️ **14 categories** — Antibiotic, Painkiller, Vitamin, Antidiabetic, Cardiac, and more
- 🌐 **Language toggle** — English only / Burmese only / Both
- 📋 **Detailed modal** — generic name, uses, dosage (adult & child), side effects
- 🎨 **Dark quantum aesthetic** with cyan/gold accents
- 📱 **Fully responsive** — works on mobile and desktop
- ⚡ **Single HTML file** — no build pipeline, no dependencies

---

## 🚀 Deploy to GitHub Pages

### Step 1 — Create a new repo
```
Repository name: medicine-dictionary
✅ Add a README file
```

### Step 2 — Upload `index.html`
Upload the file to the root of your repository.

### Step 3 — Enable GitHub Pages
```
Settings → Pages → Source: Deploy from a branch → Branch: main → / (root)
```

### Step 4 — Access your site
```
https://YOUR-USERNAME.github.io/medicine-dictionary/
```
